Year 4 Project

IoT-Based Aquaponics Monitoring System

For my Year 4 capstone project at Mapúa Malayan Colleges Laguna, I developed an IoT-based aquaponics monitoring system to optimize water quality management in a 1000-liter aquaculture setup. The system integrated Arduino and Wemos microcontrollers with sensors for pH, turbidity, dissolved oxygen, MQ137 ammonia, water level, and temperature. Using calibration techniques with standards like buffer solutions (pH 4, 7, 10) and 400 NTU formazin, I ensured accurate sensor readings. The system collected continuous data over seven days, revealing trends like ammonia spikes and high turbidity, with pH identified as a key driver of ammonia levels through regression analysis.

Key challenges included calibrating the turbidity sensor under variable indoor lighting and ensuring reliable IoT connectivity. This project enhanced my skills in IoT development, sensor calibration, Flutter programming, and full-stack integration, delivering a scalable solution for sustainable aquaponics.

Contact Support Module

During my On-the-Job Training (OJT) at Mapúa Malayan Colleges Laguna, I was trained in Flutter and Dart programming to develop a customer support module for a project, specifically contributing to Module 5-1 and Module 5-2. The module aimed to provide a responsive and intuitive interface for managing customer support requests, featuring functionalities like ticket creation, tracking, and resolution.

I designed and implemented a Flutter-based web and mobile application with a dashboard and list views, inspired by systems like Gmail, to display support tickets efficiently. The app integrated with a .NET API to handle data operations, overcoming challenges such as navigation issues and API connection errors. This project strengthened my skills in Flutter development, API integration, and user interface design, preparing me for real-world software development challenges.
